# operation-acier
jeux guerre 3d html

## Le plus simple pour débuter

Le plus simple est de passer par Render.

1. Mets le projet sur GitHub.
2. Ouvre Render et choisis `New Web Service`.
3. Connecte ton dépôt GitHub `operation-acier`.
4. Laisse Render lire [render.yaml](render.yaml).
5. Lance le déploiement.

Quand Render a fini, il te donne une adresse du style `https://ton-jeu.onrender.com`.

## En local, si tu veux tester

```bash
npm start
```

Puis ouvre http://localhost:8080
